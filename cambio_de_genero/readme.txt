# 📊 Análisis de Datos - Cambio de género en las provincias argentinas desde 2011 a 2024

## 🎯 Objetivo
Analizar edades, ubicaciones y cantidades de personas que realizaron un cambio de género en las distintas provincias del país, incluida la Ciudad de Buenos Aires.

## 🛠 Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
El dataset contiene información sobre grupo etario, género y ubicación donde se realizó el cambio.
## 📈 Análisis realizado
- Limpieza de datos
- Análisis exploratorio (EDA)
- Visualización de tendencias
- Comparación de categorías

## 🚀 Cómo ejecutar el proyecto
1. Clonar el repositorio
2. Instalar las dependencias
3. Abrir el archivo .ipynb en Jupyter Notebook
